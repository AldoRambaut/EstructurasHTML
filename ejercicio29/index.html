Notas de HTML

Principales etiquetas en HTML

Los Titulos H1, H2, H3, H4 y H5

Los titulos se escriben dentro de la etiqueta H y el numero segun el orden de importancia que tenga

Los Parrafos

Los parrafos se escriben dentro de las etiquetas "p", el parrafo se ajusta al ancho de la ventana

Enlaces o links

Se encierran en la etiqueta "a", se debe establecer a donde dirige este enlace, ya sea un punto de este documento por medio de el id, alguna pagina web, o otro archivo html

Listas

Listas Ordenadas

Se encierran con la etiqueta "ol" y cada item se encierra en la etiqueta "li"

Listas Desordenadas

Se encierran con la etiqueta "ul" y cada item se encierra en la etiqueta "li"
